Collections em Java

Collections

List

  • Características:
    • Ordem definida
    • Permite duplicados
    • Acesso por índice
  • Principais métodos:
    • add(E e)
    • get(int i)
    • remove(int i)
  • Tipos:
    • ArrayList
    • LinkedList
    • Vector
List lista = new ArrayList<>();

Set

  • Características:
    • Não tem ordem garantida
    • Não permite duplicados
    • Não garante acesso por índice
  • Principais métodos:
    • add(E e) - Retorna ‘false’ se já existir
    • remove(Object o)
    • contains(Object o)
  • Tipos:
    • HashSet
    • LinkedHashSet
    • TreeSet
Set conjunto = new HashSet<>();

Map

  • Características:
    • Pares de chave e valor
    • Chaves únicas
    • Acesso eficiente
  • Principais métodos:
    • put(K key, V value)
    • get(Object key)
    • containsKey(Object key)
  • Tipos:
    • HashMap
    • LinkedHashMap
    • TreeMap
Map mapa = new HashMap<>();
