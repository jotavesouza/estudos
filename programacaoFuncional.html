Exemplo de Ordenação

Com Lambda:

pessoaPorAltura
    .sort((p1, p2) -> Double.compare(p1.getAltura(), p2.getAltura()));

pessoaPorAltura.sort(

Funcional Interface

Interface com um Single Abstract Method (SAM). Usada com Lambda e Method References.

@FunctionalInterface - Garante que há um único método abstrato.

@FunctionalInterface
public interface Operacao {
    int calcular(int a, int b);
}

public class Main {
    public static void main(String[] args) {
        Operacao soma = (a, b) -> a + b;
        int resultado = soma.calcular(3, 5);
    }
}

Exemplos de Interfaces Funcionais

  • Consumer<T>: Aceita um valor do tipo T e não retorna nada.
  • Supplier<T>: Não aceita valores, retorna um valor do tipo T.
  • Function<T, R>: Recebe um valor do tipo T e retorna um valor do tipo R.
  • Predicate<T>: Recebe um valor do tipo T e retorna um booleano.
  • BinaryOperator<T>: Recebe dois valores do tipo T e retorna um valor do mesmo tipo.

Exemplo de Uso do Consumer:

List numeros = Arrays.asList(1, 2, 3, 4, 5);
numeros.forEach(n -> {
    if (n % 2 == 0) {
        System.out.println(n);
    }
});

Exemplo de Uso do Supplier:

List listaSaudacoes = Stream.generate(() -> "Olá, seja bem-vindo(a)!");
listaSaudacoes.forEach(System.out::println);
