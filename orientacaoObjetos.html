Orientação a Objetos em Java

Comparator X Comparable

Comparable

A lógica é definida dentro da própria classe.

Class Pessoa implements Comparable {
    // atributos
    //getter e setter

    @Override
    public int compareTo(Pessoa p) {
        return Integer.compare(this.idade, p.idade);
    }
}

Comparator

A lógica é definida fora da classe. Permite mais de um critério de ordenação ou quando não pode modificar a classe original.

Class Pessoa {
    // atributos
    // getter e setter
}

Class CompararPorIdade implements Comparator {
    @Override
    public int compare(Pessoa p1, Pessoa p2) {
        return Integer.compare(p1.getIdade(), p2.getIdade());
    }
}
