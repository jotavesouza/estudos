Spring Boot

Spring Framework

Injeção de dependência

  • Padrão de design usado para diminuir o acoplamento entre componentes
  • O Spring gerencia a criação e o ciclo de vida de objetos (beans) e injeta as dependências em tempo de execução

Spring Boot

  • Simplifica a configuração e o gerenciamento de dependências
  • Faz a configuração automática de muitos componentes, acelerando o desenvolvimento

Injeção de Dependência

  • Bean
    • O Spring cria e gerencia instâncias desses objetos e os injeta nas classes que precisam de sua funcionalidade
    • Tipos: @Component, @Service, @Controller, @Repository
  • Autowired
    • Usada para injeção automática de dependências
    • O Spring fornece automaticamente a dependência necessária (um bean) na classe que utiliza a anotação

Propriedades de Configuração

  • @Value - Permite injetar valores em campos de uma classe diretamente a partir de propriedades
  • @ConfigurationProperties - Mapear propriedades de configuração para um bean de configuração

JPA - Java Persistence API

  • ORM - Mapeamento de objetos Java para tabelas de um banco de dados relacional
  • CrudRepository, JPARepository, PagingAndSortingRepository
  • Anotações: @Query, @Param
